How to Change Admin Password

  1. Open your browser's developer console (F12)
  2. Paste this code and press Enter: sha256("your-new-password").then(hash => console.log("New Hash: " + hash));
  3. Copy the generated hash value
  4. Replace the ADMIN_PASSWORD_HASH value in the script with your new hash
Security Note: This must be done before uploading to GitHub. Never use simple passwords like "admin123".
